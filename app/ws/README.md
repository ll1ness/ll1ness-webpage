# Weather Seeker

Анимированный сайт погоды с ручным вводом города. Использует Open-Meteo API (бесплатно, без ключа).

## 🎨 Особенности дизайна

- **Liquid Glass** - современный glassmorphism в чёрно-белой гамме
- **Фиксированный интерфейс** - боковая панель + шапка + контент
- **Компактный дизайн** - оптимизировано для всех экранов
- **Адаптивность** - телефоны, планшеты, ноутбуки
- **Плавные анимации** - cubic-bezier переходы

## 🌟 Функциональность

- ✅ Ручной ввод города
- ✅ Автоопределение по геолокации (кнопка 📍)
- ✅ Текущая погода с деталями
- ✅ Прогноз на 5, 7, 10, 15 дней (до 16)
- ✅ 6 анимированных фонов (погода)
- ✅ Навигация: Текущая/Прогноз/Инфо/Факты
- ✅ Кэширование (5 минут)
- ✅ Модульная архитектура (ES6 modules)

## 🚀 Запуск

```bash
# Запустите локальный сервер
python -m http.server 8000

# Откройте в браузере
http://localhost:8000
```

**Важно**: Для геолокации требуется HTTPS или localhost.

## 📋 Как использовать

1. Откройте index.html через локальный сервер
2. Введите город в поле поиска или нажмите 📍 для геолокации
3. Переключайтесь между секциями через боковое меню
4. Выбирайте период прогноза (5, 7, 10, 15 дней)

## 🐛 Ограничения

- Геолокация требует HTTPS/localhost
- Макс. 16 дней прогноза (Open-Meteo)
- Ощущаемая температура - из текущих данных

## Особенности

- **Геолокация**: При первом запуске сайт запросит разрешение на определение местоположения
- **Прогноз**: Используется бесплатный API OpenWeatherMap, который предоставляет прогноз на 5 дней (с интервалом 3 часа). Кнопки 7, 10, 15, 30 дней показывают информационное сообщение о ограничении бесплатного тарифа
- **Анимации**: CSS-анимации, адаптирующиеся под текущую погоду (солнце, облака, дождь, снег, гроза, туман)
- **Адаптивный дизайн**: Сайт корректно отображается на мобильных устройствах

## Технологии

- HTML5
- CSS3 (анимации, flexbox, grid)
- Vanilla JavaScript (ES6+)
- OpenWeatherMap API

## Ограничения

- **Прогноз**: Open-Meteo предоставляет прогноз на максимум 16 дней бесплатно. Кнопка "30 дней" покажет информационное сообщение.
- **Геолокация**: Для работы геолокации требуется HTTPS или локальный сервер (localhost)
- **Данные**: Некоторые параметры (ощущаемая температура, давление) могут быть недоступны в реальном времени и берутся из часовых данных
- **Точность**: Геокодинг может находить не тот город, если название неоднозначно (например, "Москва" найдёт город в России)

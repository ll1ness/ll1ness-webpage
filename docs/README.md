# Документация проекта ЛИНЕСС

## Обзор

ЛИНЕСС - это современный веб-сайт для цифрового агентства, разработанный с использованием чистого HTML, CSS и JavaScript. Проект следует модульной архитектуре с разделением стилей и скриптов на независимые компоненты.

## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили с переменными, flexbox, grid, анимации
- **JavaScript (ES6+)** - модульный подход, классы, события
- **Canvas API** - для интерактивного фона
- **Fetch API** - для динамической загрузки данных
- **LocalStorage** - для хранения чата

## Ключевые особенности

### 1. Кастомный курсор
- Двойной курсор (точка + кружок)
- Анимация при наведении на интерактивные элементы
- Полностью заменяет системный курсор

### 2. Интерактивный фон
- Анимированные градиентные орбы
- Реагируют на движение мыши (параллакс)
- Реагируют на нажатия клавиш (риппл-эффекты)
- Реагируют на клики (взрыв частиц)

### 3. Адаптивный дизайн
- Полная адаптация под все устройства
- Медиа-запросы для разных размеров экрана
- Оптимизация для мобильных устройств

### 4. Динамическая загрузка
- Услуги и портфолио загружаются из JSON файлов
- Быстрая загрузка страниц
- Легкость обновления контента

### 5. Чат-бот
- Встроенный чат на странице контактов
- Локальное хранение истории
- Интерактивные ответы

### 6. Плавные анимации
- Scroll reveal эффекты
- Параллакс прокрутки
- Анимации при наведении
- Оптимизация производительности

## Настройка и запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Для разработки рекомендуется использовать локальный сервер:
   ```bash
   python3 -m http.server 8080
   ```
   или
   ```bash
   npx serve
   ```

## Браузерная совместимость
